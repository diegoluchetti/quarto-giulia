# Layout v2 — Quarto Giulia

Quarto **2,54 × 2,54 m** (6,45 m²). Móveis presenteados brancos — dimensões **estimadas** até medição in loco.

## Planta (vista de cima)

```
                    PN · janela + armário embutido
              ┌──────────────────────────────┐
              │▓▓▓▓▓▓▓▓▓▓▓│                │
              │  ARMÁRIO   │    cortina     │
PO porta ────►│            │                │◄─── PL
+ armário     │  ┌──────┐  │  ┌──────────┐  │     boiserie
              │  │CÔMODA│  │  │  BERÇO   │  │     + papel
              │  │ 85cm │  │  │  130cm   │  │     + berço
              │  └──────┘  │  └──────────┘  │
              │            │                │
              │     ○ poltrona             │
              │   (giro ~90cm)      [AC]   │
              └──────────────────────────────┘
                    PS · poltrona + AC
```

## Posicionamento

| Elemento | Parede | Medida | Notas |
|----------|--------|--------|-------|
| Berço branco | **PL** | 130 × 70 cm (est.) | Cabeceira na PL; longe do fluxo da porta |
| Boiserie + papel | **PL** | Painel ~1,40 × 1,80 m | Centralizado atrás do berço |
| Cômoda trocador | **PO** | 85 × 45 cm (est.) | Trecho livre ao lado do armário; luz natural indireta |
| Poltrona bouclé | **PS** canto PL | ~85 cm diâmetro giro | Arandela acima; mesa lateral opcional |
| Armário embutido | **PN + PO** | Existente | Zona bebê nas gavetas/prateleiras |
| AC split | **PS** | 9.000 BTU | Jato paralelo ao teto, nunca no berço |
| Interruptor | **PO** junto porta | — | Retrofit → 2 circuitos + dimmer |
| LED rodapé | Perímetro | ~9,4 m útil | Exceto vão porta; reserva 10,5 m |

## Circulação

- Corredor central livre: **≥ 70 cm** entre berço e poltrona/cômoda
- Porta PO abre sem bater berço ou cômoda
- Carrinho de bebê entra e faz giro de 180° no centro
