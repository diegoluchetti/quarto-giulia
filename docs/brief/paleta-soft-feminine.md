# Paleta — Soft Feminine Calm

Conceito atemporal para quarto feminino. Móveis presenteados **brancos** — base neutra que aceita acentos blush/lavanda.

## Cores

| Nome | Uso | Código sugerido |
|------|-----|-----------------|
| Off-white quente | 70% paredes (pintura) | Suvinil Baunilha / Coral Cremoso |
| Pêssego pálido | Alternativa base | Tons quentes baixa saturação |
| Rosa pó | Papel de parede, tricô, almofada | Pantone Rose Quartz adjacente |
| Lavanda acinzentada | Alternativa acento | Tom frio suave |
| Savana / carvalho claro | Harmonizar armário e puxadores | Referência madeira existente |
| Champagne fosco | Arandela, molduras | Metal quente discreto |

## Materiais

| Material | Onde |
|----------|------|
| Bouclé creme/rosa claro | Poltrona amamentação |
| Algodão 100% | Enxoval, cortinas voil |
| EVA autocolante + tinta | Boiserie |
| Papel vinílico lavável | Painel PL |
| Fita LED 2700K | Rodapé |

## Regras

1. Berço e cômoda **brancos** — âncoras neutras do quarto
2. Acento feminino via papel de parede, têxteis e decoração (não via tinta nos móveis)
3. Máximo 1 parede com papel + boiserie; demais lisas
4. Puxadores urso em madeira: opcionais — neutros ou substituíveis

## Moodboard textual

> Quarto de boutique suave: paredes quentes, painel com laços ou botânico blush, luz indireta no rodapé, poltrona bouclé, berço e cômoda brancos com linho areia e blush.
