# Brief — Quarto Giulia v2

Design de interiores para quarto de bebê feminino. Nascimento previsto **outubro/2026**.

## Metadados

| Campo | Valor |
|-------|-------|
| Área | 2,54 × 2,54 m (6,45 m²) |
| Apartamento | 3 quartos — Giulia no quarto quadrado **sem suíte** |
| Orçamento | Apertado — custo-benefício BR |
| Estética | Soft Feminine Calm (rosa pó / lavanda / pessego) |
| Berço / cômoda | **Presenteados** — fora do orçamento de compra |

## Requisitos por categoria

### Paredes (sistema em camadas)

1. Pintura base off-white / pêssego pálido
2. Boiserie EVA autocolante pintada (moldura decorativa)
3. Papel de parede em painel focal (dentro da boiserie)
4. Rodapé com LED 2700K dimmerizável

Ver [`sistema-paredes.md`](sistema-paredes.md).

### Iluminação

- Luz de teto (circuito 1) — funcional mamada/troca
- LED rodapé + arandela (circuito 2) — dimmer LED, luz calma
- Luz noturna USB complementar

Ver [`escopo-eletrico-led.md`](escopo-eletrico-led.md).

### Mobiliário a comprar

- Poltrona amamentação **giratória** em bouclé creme/rosa claro
- Mesa lateral de apoio (opcional)
- Decoração: quadros, tricô, organização

### Mobiliário presenteado / existente

- Berço branco clássico → [`../berco.json`](../berco.json)
- Cômoda branca (trocador) → [`../comoda.json`](../comoda.json)
- Armário embutido com zona bebê

Ver [`mobiliario-presenteado.md`](mobiliario-presenteado.md).

### Decoração

- Quadros com moldura branca (arte abstrata / blush)
- Puxadores urso em madeira: opcional trocar; decoração nova sem temática infantil forte
- Cortinas, tapete, têxteis — definir na Fase 2
