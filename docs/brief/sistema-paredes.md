# Sistema de paredes — camadas

## Composição vertical (parede destaque PL)

```
Teto
┌─────────────────────────┐
│ Boiserie EVA (moldura)  │
│  ┌───────────────────┐  │
│  │ Papel de parede   │  │
│  └───────────────────┘  │
└─────────────────────────┘
Pintura base
═══════════════════════════  Rodapé + LED 2700K (dimmer)
```

## Camadas

| # | Camada | Material |
|---|--------|----------|
| 1 | Base | Tinta acrílica PVA off-white/pêssego (~24 m²) |
| 2 | Boiserie | EVA autocolante 2–5 cm + massa acrílica + tinta |
| 3 | Painel | Papel vinílico 4–6 m² (laços/botânico blush) |
| 4 | Rodapé | Perfil alumínio + fita LED 2700K (~10,5 m) |

## Boiserie EVA — execução

1. Nivelar parede, 1 demão de tinta base
2. Colar strips EVA autocolante (moldura retangular no painel)
3. Massa nas emendas, lixa fina
4. Pintar moldura na cor da parede (ou blush 1 tom acima)
5. Aplicar papel **dentro** da moldura
6. Instalar rodapé LED por último

## Quantidades (quarto 2,54×2,54 m, pé-direito 2,70 m)

| Item | Qtd. |
|------|------|
| Tinta | ~24 m² (2 demãos) |
| EVA boiserie | 8–12 m linear |
| Papel painel | 4–6 m² |
| Perfil + fita LED | 10,5 m |

## Ordem de obra

1. Elétrica
2. Pintura base
3. Boiserie EVA + massa + pintura molduras
4. Papel de parede
5. Rodapé LED
6. Móveis
